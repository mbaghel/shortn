shortn

Usage

Pass a URL as a query to the 'api/new' path to receive a new shortened URL:
https://shortn.glitch.me/api/new?url=https://google.com
will return:
{ original_url: https://google.com, short_url: shortn.glitch.me/2 }
Visit the short URL generated to be redirected to the original URL.
(You can also enter a URL in the form above)
